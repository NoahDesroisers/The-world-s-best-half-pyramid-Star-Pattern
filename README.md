# 세상에서 가장 간단하고 정확한 별찍기

파이썬으로 하프-피라미드 별을 출력하는 가장 기초적이고 직관적인 코드입니다. 

`for`나 `while` 같은 난해한 내장 문법에 의존하지 않고, 누구나 이해할 수 있는 순수한 수학적 논리만을 사용하여 가장 정확하게 별(`*`)을 그려냅니다. 코딩을 처음 접하시는 분들도 흐름을 쉽게 따라가실 수 있습니다.

## 가장 '단순한' 특징들

* **가장 직관적인 연산:** 컴퓨터가 숫자를 이해하는 가장 원초적인 방식인 연산을 사용하여 오차 없는 덧셈과 뺄셈을 구현했습니다.
* **절대적인 정확성:** 별(`*`)을 단순히 텍스트로 출력하는 대신, 아스키코드의 근본인 42를 구하기 위해 6과 7을 곱하는 가장 확실하고 논리적인 방식을 채택했습니다.
* **안전한 기본 자료구조:** 헷갈리기 쉬운 파이썬의 내장 리스트를 쓰지 않았습니다. 누구나 아는 '연결 리스트'를 처음부터 끝까지 직접 작성하여 직관성을 극대화했습니다.
* **끊김 없는 쾌적함:** 고작 5줄의 별을 찍다가 컴퓨터가 멈추는 불상사가 없도록, 비동기 처리와 재귀 최적화를 가볍게 얹어 안정성을 챙겼습니다.
